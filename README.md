# Data Visualization with Python

## 📌 Overview
This repository showcases my ability to translate raw data into clear, actionable visual insights. It covers fundamental plotting techniques using **Matplotlib** and advanced statistical graphics using **Seaborn**, adhering to industry best practices for data storytelling.

## 📚 Topics Covered

### 1. Matplotlib Fundamentals
* **Introduction to Plotting:** Understanding the core concepts of data visualization and the Matplotlib architecture.
* **Line Plots:** Plotting single and multiple datasets, utilizing format strings for quick styling.
* **Exporting & Styling:** Customizing figure aesthetics and saving plots to external files.

### 2. Common Charts & Customizations (Matplotlib)
* **Bar Charts:** Creating vertical and horizontal variations, adding bar labels, and comparing multiple datasets.
* **Scatter & Pie Charts:** Customizing markers, handling multiple datasets, and adding text annotations.
* **Statistical Charts:** * Building Histograms with `axvline` for highlighting metrics.
  * Performing operations on Box Plots to analyze distributions and outliers.
  * Generating Stack Plots for cumulative data.

### 3. Advanced Matplotlib
* **Subplots:** Creating dashboard-style layouts with multiple charts in a single figure.
* **Modern Matplotlib:** Applying updated, modern styling to classic plots.

### 4. Seaborn (Statistical Data Visualization)
* **Introduction:** Leveraging Seaborn to create complex visualizations with less code.
* **Plot Types:** Mastering Relational, Categorical, and Distribution plots.
* **Heatmaps:** Using relational heatmaps to identify correlations within data matrices.
